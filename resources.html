Resources

This page includes a list of resources that were used to help create this website. All equations and information on the presented subjects can be found within several of these resources. If you want to learn more about orbital mechanics within a deeper level, please access these resources and more.

Textbooks

  • Curtis, H. D. (2014). Orbital mechanics for engineering students (Third ed.). Butterworth-Heinemann.

    • This textbook focuses on building the basics of orbital mechanics for engineering students. It includes topics such as two-body orbits, orbital maneuvers, and interplanetary trajectories.

  • Bate, R. R., Mueller, D. D., White, J. E., & Saylor, W. W. (2020). Fundamentals of astrodynamics. Courier Dover Publications.

    • This textbook covers the deeper fundamentals of orbital mechanics, building off the basics. It includes topics such as relative motion, three-body orbits, and orbital perturbations.

  • Vallado, D. A. (2013). Fundamentals of Astrodynamics and Applications (Fourth ed.). Microcosm Press.

    • This resource is an older textbook about astrodynamics. It is structured for teaching college-level students the basics and applications of orbital mechanics.

Online

  • Weber, B. (2023). Intro. Orbital Mechanics & Astrodynamics. https://orbital-mechanics.space/intro.html

    • This website provides college-level information about two-body and three-body orbits. This website includes equations and code for students to utilize and understand.

  • Battin, R. (2008). Astrodynamics. MIT OpenCourseWare. https://ocw.mit.edu/courses/16-346-astrodynamics-fall-2008/pages/lecture-notes/

    • This is a series of lectures provided by MIT OpenCourseWare that is open to the public. It includes topics such as the two-body problem, spacecraft transfers, and J2 perterbations.

  • NASA Jet Propulsion Laboratory. (2021). Three-Body Periodic Orbits. NASA. https://ssd.jpl.nasa.gov/tools/periodic_orbits.html

    • This website was created to show the different common three-body orbits around different celestial bodies through plotting and mathematical equations. It is possible to export these plots for external use.

  • Bolles, D. (2023). What is an orbit?. NASA Space Place. https://spaceplace.nasa.gov/orbits/en/

    • This website is a good resource for teachers looking to teach children the basics about orbits. It includes simple explanations and activities to engage a younger audience to learn about space.

  • Candey, R. (2025). SSC - 4D orbit viewer: Interactive 3D spacecraft trajectories by SPDF. NASA SPDF. https://sscweb.gsfc.nasa.gov/4dorbit/

    • This browser-based tool allows people to visualize predetermined orbits of several spacecraft and planets in a 3D viewer. The viewer allows for saved interactions that can be shared or used for presentations.

  • Park, R. (2021). Orbit Viewer. NASA Jet Propulsion Laboratory. https://ssd.jpl.nasa.gov/tools/orbit_viewer.html

    • This online tool is a resource for people to show how different spacecraft orbit the solar system through a heliocentric view. The tool requires a spacecraft ID to show how it orbits around the sun.
